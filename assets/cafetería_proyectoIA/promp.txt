🟤 script.js
// Botón que cambia el color de fondo del sitio
const botonColor = document.getElementById("colorButton");
const colores = ["#f3eee3", "#fffaf3", "#e0d4b7", "#d4a373"];
let indice = 0;

botonColor.addEventListener("click", () => {
    indice = (indice + 1) % colores.length;
    document.body.style.backgroundColor = colores[indice];
});

🚀 Cómo abrir el proyecto

Crea una carpeta llamada cafeteria-artesanal.

Guarda dentro los tres archivos (index.html, style.css, script.js).

Abre la carpeta en tu computadora.

Haz doble clic en index.html → se abrirá en tu navegador.

¡Listo! Verás un sitio cálido, profesional y animado.
