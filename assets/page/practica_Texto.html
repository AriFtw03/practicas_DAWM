Negrita y cursiva

Strong/Em

Atención: strong se utiliza para poner en negrita un texto. También se cierra.

La cursiva se realiza con em y al igual que la negrita, también se cierra.

Strong y em tienen un valor semántico. Es decir que aportan una importancia o un énfasis al texto editado.

b/i

Atención: b también se utiliza para poner en negrita un texto.

Esto está en cursiva.

i y b solo cambian el estilo visual del texto.

Línea Horizonal y saltos

hr funciona para crear una línea divisora horizontal. No se cierran.

br funciona para crear un salto de línea dentro de un párrafo. No se cierran.
