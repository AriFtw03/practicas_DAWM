Practica de CSS

Css nos ayuda/sirve para dar estilo y diseño a nuestra Página Web.

Para aplicarlo tenemos tres opciones:

1.-En linea - Código embebido
style="color:red">Linea de estilo con Código embebido
2.-Códig Interno - Código en la cabecera
El código debe ir en la cabecera del file
<style
p{
background:blue;
}
</style>
3.-Externo - File.css
link rel="stylesheet" href="estilo.cc"
Modelo caja #1:
Todo elemento HTML se presenta en pantalla se debe de tratar como una caja.
Modelo caja #2:
Todo elemento HTML se presenta en pantalla se debe de tratar como una caja.
Notas:

1.-Siempre se ejecuta de arriba hacia abajo, es decir la misma etiqueta la puede escribir dos veces y esta presentarán en pantalla el último efecto programado

2.-Tener cuidado con montar las etiquetas en el diseño

3.-Valide resetear antes de aplicar su diseño: Dependiendo de su diseño

Hipervínculos

Hipervínculos: Abrir en nueva pestaña

Atributo target="_blank": Este atributo se utiliza para especificar cómo se debe abrir el hipervínculo. Puede ser _blank, _self, _parent, o cualquier nombre diferente que no sea de los cuatro mencionados, que se considera _blank para abrir en una nueva pestaña o ventana. Abrir en una nueva pestaña

Hipervínculos: Abrimos en la misma pestaña

Los hipervínculos permiten a los usuarios navegar entre diferentes páginas web de manera sencilla. Sin embargo, no todos los navegadores permiten abrir enlaces en la misma pestaña. Para aquellos que desean mantener la navegación dentro de su sitio web o evitar que los usuarios se distraigan con otras páginas externas, es posible utilizar HTML y la propiedad target del elemento de ancla para abrir un enlace en la misma pestaña. Esto se logra estableciendo el valor «_self» para el atributo «target» del enlace. Abrir en la misma pestaña

Hipervínculos: Abrir en nueva pestaña, modo seguro

Atributo rel="noopener noreferrer": Este es el uso combinado de los atributos noopener y noreferrer, que se utilizan para mejorar la seguridad y la privacidad al abrir hipervínculos. Al usar estos atributos, se puede garantizar que los hipervínculos se abran en una nueva pestaña o ventana, proporcionando una experiencia de navegación más segura y responsable. Abrir en una nueva pestaña (Modo Seguro)
