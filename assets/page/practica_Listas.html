Práctica HTML de Listas

Las listas son un conjunto de elementos agrupados de manera vertical. Pueden ser anidadas también.

Lista desordenada

Las listas desordenadas muestran viñetas. Las viñetas pueden ser personalizadas, imágenes, íconos.

ul: Unordered list.

li: List item

Lista ordenada

Las listas ordenadas muestran número o letras. Siguen un orden.

ol: Ordered list(se enumera automáticamente).

Pasos para preparar un café

  1. Calentar el agua
  2. Agregar el café molido
  3. Esperar a que repose
  4. Servir y disfrutar

Listas con orden personalizado

Pueden ser letras ya sean mayúsuculas o minúsculas, como números romanos en mayúsuclas o minúsculas.

  1. Perro
  2. Gato
  3. Pájaro
  4. Pez

Lista de definición

Muestran pares término- definición.

dt: Definition term (término).

dd: Definition descriptión (definición).
